# OpenCode Troubleshooting

For troubleshooting common issues with OpenCode, refer to the [Troubleshooting](/docs/troubleshooting/) documentation.

## Common Issues

- Installation problems
- Configuration errors
- Provider connection issues
- Performance problems
- Feature-specific issues

## Getting Help

If you encounter issues not covered in the documentation:

1. Check the [GitHub Issues](https://github.com/anomalyco/opencode/issues)
2. Join the [Discord community](https://opencode.ai/discord)
3. Contact support through official channels

## Windows-Specific Issues

For Windows users, we recommend using [Windows Subsystem for Linux (WSL)](/docs/windows-wsl) for the best experience.

If you encounter issues on Windows:

- Try using WSL2
- Ensure you have the latest updates
- Check for compatibility issues with your terminal emulator
